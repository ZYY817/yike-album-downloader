# 一刻相册批量下载工具 - 项目总结

## 📊 项目概况

**项目名称**：一刻相册批量下载工具 (Yike Album Batch Downloader)  
**版本**：v1.0.0  
**发布日期**：2026-02-18  
**许可证**：MIT License  
**语言**：Python 3.8+

## ✨ 核心功能

1. **自动登录**：Playwright自动打开浏览器，扫码登录
2. **批量下载**：15线程并发，支持照片和视频
3. **原画质**：VIP用户可下载原始高清文件
4. **断点续传**：中断后可继续，不重复下载
5. **完整性校验**：自动验证文件大小
6. **智能整理**：按拍摄日期归类
7. **统计分析**：查看下载统计和完整性核对
